# TS 와 학습법 튜닝하기 미션 리뷰 가이드

> 이전 미션에서 진행했던 리뷰 내용은 기본적으로 포함합니다.

## 미션 안내

typescript-exercises 문제를 풀며 TypeScript 기초 문법을 익히는 미션입니다. 단, 코드 풀이만이 목표가 아닙니다. 학생은 자신의 학습 전략을 설계하고, 시도하고, 평가하고, 개선하는 루프를 반복합니다. 이 과정을 학습 로그에 기록하고, 학습에 도움이 되는 도구를 직접 만듭니다.

**산출물 3가지:**

- typescript-exercises 풀이 (최소 12번까지)
- 학습 로그 (`study-log.md`, 최소 4회 이상)
- 학습 도구 로그 (`study-tool-log.md`, 1개 이상)

## 📍 목표하는 학습 경험

- TypeScript 타입 시스템의 기초를 체득한다. 타입을 읽고, 좁히고, 조합하는 사고 방식을 경험한다.
- "어떻게 학습하는가"를 의식적으로 실험한다. 학습 전략을 세우고, 실행하고, 평가하고, 다음에 바꿔보는 메타인지 루프를 체험한다.
- AI를 학습 도우미로 활용하되, 단계적으로 의존도를 조절하는 습관을 만든다.

### 학습 로그

핵심 질문: **로그를 순서대로 읽었을 때, 학습 전략이 실제로 변했는가?**

TS 지식이 아니라 **전략의 변화 과정**에 피드백합니다. 더 좋은 학습 방법을 제안해주거나, 개선 피드백 루프를 어떻게 잘 돌릴지 제안합니다.
